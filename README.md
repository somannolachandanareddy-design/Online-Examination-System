# Online Examination System

A complete **frontend-based Online Examination System** built using **HTML, CSS, and JavaScript**.  
This project simulates a real online exam environment with authentication, timer-based exams, auto evaluation, and result display.

 Features:
 
 Authentication
- Student Login
- Admin Login (hardcoded for frontend)
- Session handling using `localStorage`

 Exam Module
- Multiple Choice Questions (MCQs)
- One question displayed at a time
- Next / Previous navigation
- Timer-based examination
- Auto-submit when time expires

Evaluation Module
- Automatic answer evaluation
- Score calculation
- Pass / Fail logic
- Result page after submission
